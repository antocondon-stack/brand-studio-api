# Fonts Directory (OFL)

Place these OFL-licensed font files in this directory:

| File | Description |
|------|-------------|
| `Inter-Regular.ttf` | Inter regular (400) |
| `Inter-Bold.ttf` | Inter bold (700) |
| `DMSerifDisplay-Regular.ttf` | DM Serif Display regular |

## Sources

- **Inter**: https://rsms.me/inter/ or https://fonts.google.com/specimen/Inter
- **DM Serif Display**: https://fonts.google.com/specimen/DM+Serif+Display

Download the TTF files and name them exactly as above. The `font_to_path` tool loads from `assets/fonts/` and will throw a clear error if a file is missing.
