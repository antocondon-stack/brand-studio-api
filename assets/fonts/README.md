# Fonts Directory

This directory should contain the following font files:

## Required Fonts

1. **Inter** - Sans-serif font
   - `Inter.ttf` (base weight, typically 400)
   - `Inter-400.ttf` (regular)
   - `Inter-700.ttf` (bold)
   - Additional weights: 100, 200, 300, 500, 600, 800, 900 (optional)

2. **DM Serif Display** - Serif display font
   - `DM Serif Display.ttf` (base weight, typically 400)
   - `DM Serif Display-400.ttf` (regular)
   - `DM Serif Display-700.ttf` (bold)
   - Additional weights: 100, 200, 300, 500, 600, 800, 900 (optional)

## Font Sources

- **Inter**: https://rsms.me/inter/
- **DM Serif Display**: https://fonts.google.com/specimen/DM+Serif+Display

## Installation

Download the font files and place them in this directory. The font loader will automatically detect available weights.
